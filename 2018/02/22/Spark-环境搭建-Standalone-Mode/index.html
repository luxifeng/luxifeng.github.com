Spark | 环境搭建 - Standalone Mode

Feb 22 2018

使用Spark提供的脚本启动一个简单的Standalone集群。

配置

  • hostname：ubuntu

  • os: Ubuntu 16.04.3 LTS (GNU/Linux 4.10.0-40-generic x86_64)

  • java version: 1.8.0_151

  • spark version: 2.2.1

启动步骤

1.上传安装包spark-2.2.1-bin-hadoop2.7.tgz到/home/lucy/Documents目录下

2.解压压缩包并移动到安装目录下

3.启动master

4.启动worker

Master web UI

默认URL是localhost:8080
